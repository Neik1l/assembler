# Язык ассемблера УВМ

Формат: CSV с заголовком `opcode,operand`.

Поддерживаемые команды:

| Opcode  | Описание            | Operand               |
|---------|---------------------|-----------------------|
| LOAD    | Загрузка константы  | Целое число (0–2^31-1)|
| READ    | Чтение из памяти    | —                     |
| WRITE   | Запись в память     | Адрес (0–4095)        |
| BSWAP   | Обмен байтов        | —                     |
